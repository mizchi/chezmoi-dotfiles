# E4023

## Contents

# E4023[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4023.html#e4023 "Link to this heading")

The trait is not found.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4023.html#erroneous-example "Link to this heading")

```
let a = true as &Sh0w
```

The example above tries to cast a boolean value to an object
of the `Sh0w` trait, but this trait is not found in the current scope,
giving the following error on line 1:

```
The trait Sh0w is not found.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4023.html#suggestion "Link to this heading")

Make sure the trait name is correct:

```
let a = true as &Show
```
