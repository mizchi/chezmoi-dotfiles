# E4009

## Contents

# E4009[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4009.html#e4009 "Link to this heading")

Match function expects a different number of arguments than provided.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4009.html#erroneous-example "Link to this heading")

```
let f = fn {
  0 => 0
  a, b => a + b
}
```

The example defines a match function that has incoherent number of arguments in
different branches. This will result in the following error on line 3:

```
Match function expects 1 arguments, but 2 arguments are provided.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4009.html#suggestion "Link to this heading")

Change the branches so that the number of arguments are exactly the same in all
of them:

```
let f = fn {
  0 => 0
  a => a
}
```
