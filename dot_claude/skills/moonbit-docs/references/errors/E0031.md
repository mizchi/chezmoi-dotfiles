## Contents

# E0031[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0031.html#e0031 "Link to this heading")

The optional argument is never supplied. If the optional argument is never
supplied, then it can be safely removed from the parameter list and be used
purely internally as a normal variable.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0031.html#erroneous-example "Link to this heading")

Note, this warning is turned off by default. To enable this warning, you have to
modify the `"warn-list"` of `moon.pkg.json`:

moon.pkg.json[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0031.html#id1 "Link to this code")

```
{
  "is_main": true,
  "warn-list": "+31"
}
```

main.mbt[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0031.html#id2 "Link to this code")

```
///|
fn f(opt~ : Int = 4) -> Int {
  ignore(opt)
  0
}

///|
fn main {
  println(f())
  println(f())
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0031.html#suggestion "Link to this heading")

You can remove the optional argument, and use a local variable definition
instead:

```
///|
fn f() -> Int {
  let opt = 4
  ignore(opt)
  0
}

///|
fn main {
  println(f())
  println(f())
}
```
