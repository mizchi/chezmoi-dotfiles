## Contents

# E4063[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4063.html#e4063 "Link to this heading")

Type does not implement trait, although an `impl` is defined.

This error can be further divided into 5 cases:

* Method is missing
* Implementation is private
* Method type mismatch
* Constraints are not satisfied
* Method contains unresolved type parameters

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4063.html#erroneous-example "Link to this heading")

### Method is missing[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4063.html#method-is-missing "Link to this heading")

```
trait Number {
  to_int(Self) -> Int
  to_float(Self) -> Float
}

type A Int

impl Number for A with to_int(self : A) -> Int {
//^~~~~~~~~~~~~~~
// Error: Type A does not implement trait Number, although an `impl` is defined.
// hint:
//   method to_float is missing.
  self.inner()
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4063.html#suggestion "Link to this heading")

Modify the code according to the hint provided along with the error message. For
the example above, you can add the missing method `to_float` to type `A`.

```
impl Number for A with to_float(self : A) -> Float {
  self.inner().to_float()
}
```
