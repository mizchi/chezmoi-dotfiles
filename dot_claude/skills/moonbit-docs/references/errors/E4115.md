## Contents

# E4115[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4115.html#e4115 "Link to this heading")

Cannot auto-fill parameter of this type.

MoonBit supports and auto-filling `SourceLoc` and `ArgsLoc` parameters in
functions. Leaving the default value of parameter of other types will result in
this error.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4115.html#erroneous-example "Link to this heading")

```
fn f(parameter~ : Int = _) -> Unit {
  // ^~~~~~~~~ Error: Cannot auto-fill parameter of type Int
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4115.html#suggestion "Link to this heading")

Use a default value for the parameter:

```
fn f(parameter~ : Int = 0) -> Unit {
}
```
