# E4096[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4096.html#e4096 "Link to this heading")

String interpolation syntax is not supported for `Bytes`.
Only `String` supports string interpolation.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4096.html#erroneous-example "Link to this heading")

```
test {
  let x : Int = 42
  let _ : Bytes = "(\{x})"
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4096.html#suggestion "Link to this heading")

To dynamically create a `Bytes`, use `@buffer.T` instead:

```
test {
  let x : Int = 42
  let buf = @buffer.new()
  let _ : Bytes = buf
    ..write_string("(")
    ..write_object(x)
    ..write_string(")")
    .to_bytes()
}
```
