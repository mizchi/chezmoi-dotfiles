# E4119[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4119.html#e4119 "Link to this heading")

This function is not a toplevel function, so it cannot have labelled arguments.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4119.html#erroneous-example "Link to this heading")

```
fn main {
  fn greet(name~ : String) -> Unit {
    println("Hello, \{name}")
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4119.html#suggestion "Link to this heading")

You can either remove the labelled argument, using positional argument instead:

```
fn main {
  fn greet(name : String) -> Unit {
    println("Hello, \{name}")
  }
  greet("world")
}
```

Or you can make the function a toplevel function:

```
fn greet(name~ : String) -> Unit {
  println("Hello, \{name}")
}

fn main {
  greet(name="World")
}
```
