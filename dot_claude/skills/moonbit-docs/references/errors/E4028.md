# E4028

## Contents

# E4028[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4028.html#e4028 "Link to this heading")

This expression has a type which is not a struct.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4028.html#erroneous-example "Link to this heading")

```
struct T { a : Int }
let a : Int = { a: 42 }
```

The example above tries to assign a struct of type `T` to a variable `a` of type `Int`,
which is not possible and gives the following error on line 2:

```
This expression has type Int, which is a Int type and not a struct.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4028.html#suggestion "Link to this heading")

Make sure to use the correct type instead:

```
struct T { a : Int }
let a : T = { a: 42 }
```
