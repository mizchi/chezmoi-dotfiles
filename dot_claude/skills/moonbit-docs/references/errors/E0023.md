# E0023[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0023.html#e0023 "Link to this heading")

The body of this try expression never raises any error.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0023.html#erroneous-example "Link to this heading")

```
///|
fn main {
  println("Hello, world!") catch { // `try` is omitted here
    _ => println("Error")
  }
  try {
    println("Hello, ")
    println("world!")
  } catch {
    _ => println("Error in try block")
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0023.html#suggestion "Link to this heading")

Remove the `try` expression:

```
///|
fn main {
  println("Hello, world!")
}
```
