# E4154[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4154.html#e4154 "Link to this heading")

Non-constant `enum` with custom tag value.

MoonBit supports customizing the integer representation of constructors:

```
enum Flag {
  A = 1
  B = 2
  C = 3
}
```

However, only constant `enum` (all constructors have not payload) can have custom tag.
Otherwise this error will be reported.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4154.html#erroneous-example "Link to this heading")

```
enum Bad {
  A = 1
  B = 2
  C(Int)
}
```
