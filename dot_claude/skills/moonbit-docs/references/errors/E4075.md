# E4075[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4075.html#e4075 "Link to this heading")

Missing type annotation for the parameter.

MoonBit requires all toplevel function to have full type annotation.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4075.html#erroneous-example "Link to this heading")

```
fn f(param) -> Unit { // Error: Missing type annotation for the parameter param.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4075.html#suggestion "Link to this heading")

Add type annotation for the parameter:

```
fn f(param: Int) -> Unit {
}
```

Or if the parameter should be generic:

```
fn f[T](param: T) -> Unit {
}
```
