# E4044

## Contents

# E4044[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4044.html#e4044 "Link to this heading")

Struct fields are missing. Use `..` to ignore them in patterns.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4044.html#erroneous-example "Link to this heading")

```
struct S { a : Int; b: Int }
let a : Int = match S::{ a: 2, b: 3 } {
  { a: 2 } => 4
  _ => 6
}
```

The example above tries to match a struct with a missing field `b`,
giving the following error on line 3:

```
Struct fields b are unmatched, use `..` to ignore them.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4044.html#suggestion "Link to this heading")

Make sure to provide all fields in the pattern,
or simply ignore the missing fields using `..`:

```
struct S { a : Int; b: Int }
let a : Int = match S::{ a: 2, b: 3 } {
  { a: 2, b: 3 } => 4
  { a: 2, .. } => 5
  _ => 6
}
```
