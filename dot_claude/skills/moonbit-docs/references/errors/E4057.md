# E4057[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4057.html#e4057 "Link to this heading")

The constructor is duplicate.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4057.html#erroneous-example "Link to this heading")

```
enum E {
  A
  A(Int)
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4057.html#suggestion "Link to this heading")

Rename the constructor to a different name.

```
enum E {
  A
  B(Int)
}
```
