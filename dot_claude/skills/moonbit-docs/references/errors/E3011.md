# E3011

## Contents

# E3011[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3011.html#e3011 "Link to this heading")

The assignment contains an invalid left-hand-side (LHS) expression,
such as a constant or a constructor.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3011.html#erroneous-example "Link to this heading")

```
const N = 4

fn main {
  N = 5 // Error: Invalid left value for assignment.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3011.html#suggestion "Link to this heading")

Change the LHS to a valid mutable memory location,
such as a mutable variable or a mutable field:

```
fn main {
  let mut n = 4
  n = 5
}
```
