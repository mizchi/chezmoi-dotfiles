# E4147

## Contents

# E4147[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4147.html#e4147 "Link to this heading")

Range pattern bounds must satisfy the ordering constraints:

| Pattern | Constraint |
| --- | --- |
| `a..=b` | `a <= b` |
| `a..<b` | `a < b` |

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4147.html#erroneous-example "Link to this heading")

```
fn main {
  let value = 1
  match value {
    0..<-10 => println("0..<-10")
  //^~~~~~~
  // Error: Range pattern `a..<b` must satisfy `a < b`.
    _ => println("_")
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4147.html#suggestion "Link to this heading")

Make sure the range pattern bounds satisfy the ordering constraints. This can be
usually achieved by swapping the bounds.

```
fn main {
  let value = 1
  match value {
    -10..<0 => println("0..=10")
    _ => println("_")
  }
}
```
