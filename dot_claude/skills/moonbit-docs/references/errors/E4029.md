# E4029

## Contents

# E4029[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4029.html#e4029 "Link to this heading")

This expression has a type which is not a variant.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4029.html#erroneous-example "Link to this heading")

```
fn main {
  fn f {
    (a, b) => a + b
  }
  let a = f(None)
}
```

The example above tries to call a function `f` with a value `None`,
which is a variant of the `Option[_]` type, but `f` only accepts a tuple,
giving the following error on line 5:

```
This expression has type (Int, Int), which is a tuple type and not a variant.
```

## Suggestions[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4029.html#suggestions "Link to this heading")

Make sure to use a value of the correct type instead:

```
fn main {
  fn f {
    (a, b) => a + b
  }
  let a = f((3, 4))
}
```
