# E0055[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0055.html#e0055 "Link to this heading")

Unannotated FFI param type.

The FFI parameter of pointer type should be annotated with either `#borrow` or
`#owned`, the default calling convention is currently `#owned` and it will be
changed in the future. For the detailed meaning, checkout
[FFI documentation](https://docs.moonbitlang.com/en/stable/language/ffi.html#the-borrow-and-owned-attribute).

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0055.html#erroneous-example "Link to this heading")

```
///|
extern "c" fn _f(p : Bytes) -> Unit = "free"
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0055.html#suggestion "Link to this heading")

Add the missing `#borrow` or `#owned` annotation.

```
///|
#borrow(p)
extern "c" fn _f(p : Bytes) -> Unit = "free"
```
