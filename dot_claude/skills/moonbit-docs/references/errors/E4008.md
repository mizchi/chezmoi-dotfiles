# E4008[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4008.html#e4008 "Link to this heading")

FFI function cannot have type parameters.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4008.html#erroneous-example "Link to this heading")

```
extern "js" fn id[T](x: T) -> T = "(x) => x"
```

The example declares an FFI function (marked `extern`) with a type parameter,
which is not allowed.

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4008.html#suggestion "Link to this heading")

Consider using a concrete type that suits your needs:

```
extern "js" fn int_id(x: Int) -> Int = "(x) => x"
```

For more complicated scenarios, consider adding an extra trait:

```
///|
trait Ider {
  id(Self) -> Self
}

///|
impl Ider for Int with id(self) { int_id(self) }
```
