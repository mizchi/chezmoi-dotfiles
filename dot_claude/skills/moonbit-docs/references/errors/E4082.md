## Contents

# E4082[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4082.html#e4082 "Link to this heading")

Variable is not bound in all patterns.

When using the `|` operator in a pattern, all variables must be bound in all
patterns. If a variable were not bound in all patterns, it would be free
variable when the pattern is matched, which is not allowed.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4082.html#erroneous-example "Link to this heading")

```
enum E {
  A(Int, Double)
  B(Int)
}

fn f(value : E) -> Unit {
  match value {
    A(a, _) | B(_) => println("Hello") // Error: Variable a is not bound in all patterns.
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4082.html#suggestion "Link to this heading")

```
fn f(value : E) -> Unit {
  match value {
    A(a, _) | B(a) => println("Hello")
  }
}
```
