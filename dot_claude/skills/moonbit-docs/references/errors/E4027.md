# E4027[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4027.html#e4027 "Link to this heading")

Unused type parameter.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4027.html#erroneous-example "Link to this heading")

```
fn id[T](a: Int) -> Int { a }
```

The example above declares a type parameter `T` in the identity function `id`,
however `T` is not used anywhere in the function signature,
giving the following error on line 1:

```
Unused type parameter 'T'
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4027.html#suggestion "Link to this heading")

Make sure the type parameter is used in the function signature:

```
fn id[T](a: T) -> T { a }
```
