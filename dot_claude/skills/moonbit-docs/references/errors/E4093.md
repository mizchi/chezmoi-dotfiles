# E4093

## Contents

# E4093[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4093.html#e4093 "Link to this heading")

The type is not a struct type.

This error occurs when you try to construct a type that is not a `struct` using
the `T::{ .. }` syntax.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4093.html#erroneous-example "Link to this heading")

```
enum Point {
  D2(Double, Double)
  D3(Double, Double, Double)
}

fn main {
  let a = Point::{ x : 1.0, y : 2.0 }
  //      ^~~~~
  // Error: The type Point is not a struct type
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4093.html#suggestion "Link to this heading")

You should use the correct syntax to construct the type.

```
fn main {
  let a = Point::D2(1.0, 2.0)
}
```
