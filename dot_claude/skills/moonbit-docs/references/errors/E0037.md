# E0037

## Contents

# E0037[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0037.html#e0037 "Link to this heading")

The loop label name is never used.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0037.html#erroneous-example "Link to this heading")

```
///|
fn main {
  let mut sum = 0
  let input = [1, 2, 3]
  read~: loop input[:] {
    [] => break
    [head, .. tail] => {
      sum += head
      continue tail
    }
  }
  println(sum)
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0037.html#suggestion "Link to this heading")

Remove the unused loop label.

```
///|
fn main {
  let mut sum = 0
  let input = [1, 2, 3]
  loop input[:] {
    [] => break
    [head, .. tail] => {
      sum += head
      continue tail
    }
  }
  println(sum)
}
```
