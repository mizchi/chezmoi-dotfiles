# E4077

## Contents

# E4077[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4077.html#e4077 "Link to this heading")

Don't know how to derive trait for type.

MoonBit allows you to derive implementation of some pre-defined traits for your
types. It means that you cannot have MoonBit automatically derive a trait that
you defined yourself.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4077.html#erroneous-example "Link to this heading")

```
trait T {
  f(Self) -> Int
}

type A Int derive(T) // Error: Don't know how to derive trait T for type A
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4077.html#suggestion "Link to this heading")

You can implement the trait manually:

```
trait T {
  f(Self) -> Int
}

// Remove derive(T)
type A Int

impl T for A with f(self : A) -> Int {
  0
}
```
