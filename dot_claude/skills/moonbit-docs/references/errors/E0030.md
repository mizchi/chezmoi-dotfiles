# E0030

## Contents

# E0030[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0030.html#e0030 "Link to this heading")

The package alias is empty. The default package alias will be used instead.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0030.html#erroneous-example "Link to this heading")

moon.pkg.json[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0030.html#id1 "Link to this code")

```
{
  "import": [
    {
      "path": "moonbit-community/E0030/lib",
      "alias": ""
    }
  ]
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0030.html#suggestion "Link to this heading")

Use a non-empty alias for the package:

moon.pkg.json[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0030.html#id2 "Link to this code")

```
{
  "import": [
    {
      "path": "moonbit-community/E0030-fixed/lib",
      "alias": "hello"
    }
  ]
}
```
