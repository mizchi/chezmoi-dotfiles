# E4099[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4099.html#e4099 "Link to this heading")

A trait is used in context where a type is needed.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4099.html#erroneous-example "Link to this heading")

```
// `Show` is a trait, not a type
fn f(_ : Show) -> Unit {

}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4099.html#suggestion "Link to this heading")

If you want to refer to a trait object type, use the syntax `&Trait` instead.

```
fn f(_ : &Show) -> Unit {

}
```
