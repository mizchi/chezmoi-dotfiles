# E4114

## Contents

# E4114[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4114.html#e4114 "Link to this heading")

Only toplevel functions can have labelled arguments.

This error occurs when trying to use labelled arguments in a local function or
anonymous function.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4114.html#erroneous-example "Link to this heading")

```
pub fn f() -> Unit {
  fn h(x~: Int) -> Unit {
//     ^^ Error: Only toplevel functions can have labelled arguments.
    println(x)
  }
  h(x=42)
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4114.html#suggestion "Link to this heading")

To fix this error, you can either:

* Move the function to the top level of the module

```
fn h(x~: Int) -> Unit {
  println(x)
}

pub fn f() -> Unit {
  h(x=42)
}
```

* Remove the labelled arguments

```
pub fn f() -> Unit {
  fn h(x: Int) -> Unit {
    println(x)
  }
  h(42)
}
```
