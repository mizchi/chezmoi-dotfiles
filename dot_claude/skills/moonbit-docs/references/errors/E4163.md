# E4163

## Contents

# E4163[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4163.html#e4163 "Link to this heading")

Local parameters are not local.

`#locals` is an attribute that aims to inline the given function parameter. This
error is raised when the given parameter is not a 'local' one.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4163.html#erroneous-example "Link to this heading")

```
///|
#locals(f)
pub fn[A, B] map(
  arr : ArrayView[A],
  f : (A) -> B,
  target? : Array[B] = [],
) -> Array[B] {
  match arr {
    [hd, .. tl] => {
      target.push(f(hd))
      map(tl, f, target~) // The parameter is not local: it escapes.
    }
    [] => target
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4163.html#suggestion "Link to this heading")

Apply `locals` to local functions only.

```
///|
#locals(f)
pub fn[A, B] map(
  arr : ArrayView[A],
  f : (A) -> B,
  target? : Array[B] = [],
) -> Array[B] {
  loop arr {
    [hd, .. tl] => {
      target.push(f(hd))
      continue tl
    }
    [] => target
  }
}
```
