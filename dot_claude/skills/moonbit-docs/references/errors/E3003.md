## Contents

# E3003[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3003.html#e3003 "Link to this heading")

`init` and `main` function must have no arguments and no return value.

Note

The case of `fn main() -> Unit` is exceptionally accepted
and can be corrected by the formatter.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3003.html#erroneous-example "Link to this heading")

```
///|
/// `init` function must have no arguments and no return value.
fn init() -> Unit {

}

///|
/// Error: Unused parameter list for the main function.
fn main() {
  println("Hello, world!")
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3003.html#suggestion "Link to this heading")

Remove the argument list and return type annotation, as:

```
///|
fn init {

}

///|
fn main {
  println("Hello, world!")
}
```
