## Contents

# E4124[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4124.html#e4124 "Link to this heading")

The constructor is ambiguous: it may come from multiple types.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4124.html#erroneous-example "Link to this heading")

```
enum A {
  A(Int)
  B(Bool)
  C(Double)
}

enum B {
  A(Double)
  B(Int)
  C(Bool)
}

fn main {
  let a = A(1) // Error: The constructor A is ambiguous: it may come from type B or A.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4124.html#suggestion "Link to this heading")

Add `T::` before the constructor:

```
fn main {
  let a = A::A(1)
}
```
