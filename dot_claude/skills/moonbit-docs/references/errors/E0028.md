# E0028[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0028.html#e0028 "Link to this heading")

Unfinished code. In MoonBit, you can use `...` to mark a piece of code that is incomplete. You can still compile and run the code, but it will cause a runtime error when it is executed.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0028.html#erroneous-example "Link to this heading")

```
///|
fn f() -> Unit {
  ...
}
```
