# E3002[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3002.html#e3002 "Link to this heading")

This source files contains errors in the syntax of the code.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3002.html#erroneous-example "Link to this heading")

```
fn main() -> Unit {
  println("Hello, world!"
}
```

This example gives the following error on line 3:

```
Parse error, unexpected token `}`, you may expect `,` or `)`.
```

... which indicates a missing closing parenthesis (`)`) in the `println` function call.

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3002.html#suggestion "Link to this heading")

Change your code to strictly follow the MoonBit syntax rules.

In the above example, the missing closing parenthesis should be added:

```
fn main {
  println("Hello, world!")
}
```
