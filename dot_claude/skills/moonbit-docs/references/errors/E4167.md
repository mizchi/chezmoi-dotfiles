## Contents

# E4167[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4167.html#e4167 "Link to this heading")

Default implementation is not marked with `= _`.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4167.html#erroneous-example "Link to this heading")

```
///|
pub(open) trait HasDefaultImpl {
  foo() -> Unit // It has default implementation but is not marked
}

///|
impl HasDefaultImpl with foo() {
  println("OK")
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4167.html#suggestion "Link to this heading")

Add `= _` to the methods that have default implementations.

```
///|
pub(open) trait HasDefaultImpl {
  foo() -> Unit = _
}
```
