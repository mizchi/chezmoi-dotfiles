# E4128[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4128.html#e4128 "Link to this heading")

Target of type alias must not be a type parameter.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4128.html#erroneous-example "Link to this heading")

```
typealias Wrap[T] = T // Error: Target of type alias must not be a type parameter.
```
