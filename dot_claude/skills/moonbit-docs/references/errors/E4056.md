# E4056

## Contents

# E4056[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4056.html#e4056 "Link to this heading")

Method has already been defined.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4056.html#erroneous-example "Link to this heading")

```
struct Point {
  x: Int
  y: Int
}

fn Point::to_string(self : Point) -> String {
  "(" + self.x.to_string() + "," + self.y.to_string() + ")"
}

fn Point::to_string(self : Point) -> String {
  "<" + self.x.to_string() + "," + self.y.to_string() + ">"
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4056.html#suggestion "Link to this heading")

Remove the duplicate method and keep only one method:

```
struct Point {
  x: Int
  y: Int
}

fn Point::to_string(self : Point) -> String {
  "(" + self.x.to_string() + "," + self.y.to_string() + ")"
}

// Remove the second Point::to_string method
```
