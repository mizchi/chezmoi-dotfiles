## Contents

# E4158[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4158.html#e4158 "Link to this heading")

Default implementation is required but not found.

When a trait declares that it has a default implementation, the implementation
should exist.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4158.html#erroneous-example "Link to this heading")

```
///|
pub(open) trait Close {
  // A default implementation is required but not found.
  close(Self) -> Unit raise = _
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4158.html#suggestion "Link to this heading")

Implement the default implementation.

```
///|
impl Close with close(_) {
  ()
}
```
