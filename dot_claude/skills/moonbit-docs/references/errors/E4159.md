## Contents

# E4159[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4159.html#e4159 "Link to this heading")

Implementations are missing.

When a package implements a
[virtual package](https://docs.moonbitlang.com/en/stable/toolchain/moon/package.html#virtual-package), it is checked
against the interface of that virtual package, and this error is emitted if some
of the interface is missing.

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4159.html#suggestion "Link to this heading")

Implement the missing implementations and fulfill the interface. Make sure that
the expected definitions are indeed public.

```
///|
pub fn f() -> Unit {

}

///|
pub fn g() -> Unit {

}
```
