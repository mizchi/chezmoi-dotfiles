# E4091

## Contents

# E4091[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4091.html#e4091 "Link to this heading")

The type has no field with the given name.

It is possible that you made a typo and use a wrong field name to access the
field of a struct.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4091.html#erroneous-example "Link to this heading")

```
struct Point {
  x : Double
  y : Double
}

fn main {
  let point = Point::{ x : 1.0, y : 2.0 }
  println(point.z) // Error: The type Point has no field z.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4091.html#suggestion "Link to this heading")

You can either add the missing field to the struct, or use an existing field
to access the struct.

```
struct Point {
  x : Double
  y : Double
}

fn main {
  let point = Point::{ x : 1.0, y : 2.0 }
  println(point.x)
  println(point.y)
}
```
