# E4014[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4014.html#e4014 "Link to this heading")

Type Mismatch.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4014.html#erroneous-example "Link to this heading")

```
fn fst[X, Y] (a: X, b: Y) -> X {
  b
}
```

The above example returns a value `b` of type `Y` on line 2,
which is not the same as the return type `X` of the function.

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4014.html#suggestion "Link to this heading")

Please make sure to return a value of the same type as the return type
of the function.

For instance, in the example above, you may write:

```
fn fst[X, Y] (a: X, b: Y) -> X {
  a
}
```
