# E4012[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4012.html#e4012 "Link to this heading")

Mutable constructor fields are only allowed on labelled arguments.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4012.html#erroneous-example "Link to this heading")

```
enum E {
  F(mut x : Int)
}
```

The example above declares an unlabeled mutable field `x` in the
constructor of the `F` variant of the `E` enum, which is not allowed.

## Suggestions[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4012.html#suggestions "Link to this heading")

Add a label to the mutable field:

```
enum E {
  F(mut x~ : Int)
}
```
