# E0035[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0035.html#e0035 "Link to this heading")

The word is reserved for possible future use. Please use another name for it.

Check [here](https://docs.moonbitlang.com/en/stable/language/introduction.html#reserved-keywords) for the full list of
reserved keywords

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0035.html#erroneous-example "Link to this heading")

```
///|
/// Warning: The word `module` is reserved for possible future use.
/// Please consider using another name.
fn module() -> Int {
  3
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0035.html#suggestion "Link to this heading")

```
///|
fn module_() -> Int {
  3
}
```
