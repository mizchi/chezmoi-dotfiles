# E4015

## Contents

# E4015[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4015.html#e4015 "Link to this heading")

Type has no method with the specified name.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4015.html#erroneous-example "Link to this heading")

```
let hey = "hey".upper()
```

The above example tries to call an inexistent method `upper()`
on a string literal, giving the following error on line 1:

```
Type String has no method upper.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4015.html#suggestion "Link to this heading")

This is usually a typo or a misunderstanding of the methods available.
Please make sure to use the correct method name:

```
let hey = "hey".to_upper()
```
