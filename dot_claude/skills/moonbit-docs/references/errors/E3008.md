## Contents

# E3008[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3008.html#e3008 "Link to this heading")

There are multiple `..` patterns in array pattern. Remove until there is only one `..` pattern in array pattern.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3008.html#erroneous-example "Link to this heading")

```
fn main {
  let array = [1, 2, 3, 4, 5]
  let [fst, .., .., snd] = array
  //            ^^
  // Error: At most one `..` is allowed in array pattern.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3008.html#suggestion "Link to this heading")

Remove the extra `..` pattern.

```
fn main {
  // ...
  let [fst, .., snd] = array
}
```
