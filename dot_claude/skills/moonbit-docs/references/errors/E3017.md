# E3017[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3017.html#e3017 "Link to this heading")

JSON parse error. This will only appear in `moon.pkg.json` files only.

It is very like that the editor you are using will have support for JSON
installed (for example, Visual Studio Code has a built-in JSON parser). This
error is likely to be caused by a missing comma or a missing closing brace.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3017.html#erroneous-example "Link to this heading")

```
{
  "is-main": true
  "import": [] // Error: Expect_comma_or_rbrace
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3017.html#suggestion "Link to this heading")

```
{
  "is-main": true,
  "import": []
}
```
