## Contents

# E4055[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4055.html#e4055 "Link to this heading")

Field is already declared.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4055.html#erroneous-example "Link to this heading")

```
struct A {
  a : Int
  a : Double
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4055.html#suggestion "Link to this heading")

Rename the field to a different name.

```
struct A {
  a : Int
  b : Double
}
```
