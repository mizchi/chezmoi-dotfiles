## Contents

# E4140[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4140.html#e4140 "Link to this heading")

Invalid C function name in extern "C" declaration.

When binding a C function with the `extern "C"` declaration, the function name
must be a valid C identifier. This means that the name must satisfy the
following regex:

```
[a-zA-Z_$][a-zA-Z0-9_$]*
```

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4140.html#erroneous-example "Link to this heading")

```
extern "C" fn f1() = "1" // Error: Invalid C function name in extern "C" declaration
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4140.html#suggestion "Link to this heading")

Change the function name to a valid C identifier:

```
extern "C" fn f1() = "f1"
```
