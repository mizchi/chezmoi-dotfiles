## Contents

# E0024[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0024.html#e0024 "Link to this heading")

The error type of this function is never used.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0024.html#erroneous-example "Link to this heading")

```
///|
fn greet() -> Unit raise {
  println("Hello, world!")
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0024.html#suggestion "Link to this heading")

Remove the error type:

```
///|
fn greet() -> Unit {
  println("Hello, world!")
}
```
