## Contents

# E4007[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4007.html#e4007 "Link to this heading")

Enum variants without payloads are considered as singleton values. MoonBit does
not allow such constructors to be called as functions.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4007.html#erroneous-example "Link to this heading")

```
enum Status {
  Done
  Pending
}

fn main {
  let status = Done() // Error: constructor without payload cannot be called with ()
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4007.html#suggestion "Link to this heading")

Remove the function calling syntax from the constructor:

```
// ...
fn main {
  let status = Done
}
```

If you really need the constructor to construct values, explicitly use `Unit` as
its argument

```
enum Status {
  Done(Unit)
  Pending
}

fn main {
  let status = Done(())
}
```
