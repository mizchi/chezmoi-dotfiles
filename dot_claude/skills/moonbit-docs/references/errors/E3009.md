# E3009[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3009.html#e3009 "Link to this heading")

Struct pattern cannot contain only `..`, use wildcard pattern `_` instead.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3009.html#erroneous-example "Link to this heading")

```
struct Point {
  x: Int
  y: Int
}

fn process_point(p: Point) -> Unit {
  match p {
    { .. } => println("Got a point")
  //^~~~~~
  // Error: Struct pattern cannot contain only `..`, use wildcard pattern `_` instead.
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3009.html#suggestion "Link to this heading")

Use the wildcard pattern `_` instead of `{ .. }`:

```
struct Point {
  x: Int
  y: Int
}

fn process_point(p: Point) -> Unit {
  match p {
    _ => println("Got a point")
  }
}
```

You can also use `{ .. }` along with other fields if you want to match specific fields:

```
fn process_point(p: Point) -> Unit {
  match p {
    { x: 0, .. } => println("Point on y-axis")
    _ => println("Other point")
  }
}
```
