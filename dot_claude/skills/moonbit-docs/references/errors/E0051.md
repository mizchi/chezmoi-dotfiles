# E0051[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0051.html#e0051 "Link to this heading")

Ambiguous operator precedence.

Some operators have non-intuitive precedence and can lead to confusion. Always
use parentheses to make the intent clearer in these cases.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0051.html#erroneous-example "Link to this heading")

```
///|
pub fn f() -> Int {
  1 + 2 << 3
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0051.html#suggestion "Link to this heading")

Add parentheses to make the intent clearer.

```
///|
pub fn f() -> Int {
  (1 + 2) << 3
}
```
