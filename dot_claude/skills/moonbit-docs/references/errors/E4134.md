# E4134

## Contents

# E4134[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4134.html#e4134 "Link to this heading")

The return type of this anonymous function is expected to include an error type.
Please add the error type to the return type annotation or use `fn!` instead.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4134.html#erroneous-example "Link to this heading")

```
fn main {
  let draw : (Int) -> Int! = fn(luck : Int) -> Int {
    //                                         ^~~
    // Error: The return type of this anonymous function is expected include an
    // error type. Please add the error type to the return type annotation or
    // use `fn!` instead.
    if luck == 7 {
      return 42
    }
    fail!("Bad luck") // E4122 as well
  }
  try {
    println("Draw: \{draw!(7)}")
  } catch {
    error => println("Error: \{error}")
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4134.html#suggestion "Link to this heading")

Add the error type to the return type annotation:

```
fn main {
  let draw : (Int) -> Int! = fn(luck : Int) -> Int! {
    if luck == 7 {
      return 42
    }
    fail!("Bad luck")
  }
  try {
    println("Draw: \{draw!(7)}")
  } catch {
    error => println("Error: \{error}")
  }
}
```
