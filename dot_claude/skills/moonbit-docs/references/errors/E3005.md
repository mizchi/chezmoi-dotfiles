## Contents

# E3005[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3005.html#e3005 "Link to this heading")

There is no such visibility for the entity (function/type/trait/...).

Usually, this means that you put an `priv` visibility modifier on a entity is
by-default private.

See the [Access Control](https://docs.moonbitlang.com/en/stable/language/packages.html#access-control) section of for a
detailed explanation on the visibility in MoonBit.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3005.html#erroneous-example "Link to this heading")

```
priv let value = 3 // Error: No 'priv' visibility for value.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3005.html#suggestion "Link to this heading")

Remove the visibility modifier from the definition of the entity:

```
let value = 3 // This is already `priv` by default.
```
