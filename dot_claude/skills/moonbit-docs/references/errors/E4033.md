# E4033

## Contents

# E4033[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4033.html#e4033 "Link to this heading")

There is no struct definition with the specified fields.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4033.html#erroneous-example "Link to this heading")

```
fn main {
  struct S { x : Int; y : Int }
  let c = { x: 2, w: 1 }
}
```

The example above tries to assign a struct with fields `x` and `w` to a variable `c`,
but this field doesn't exist in any known struct type, giving the following error on line 3:

```
There is no struct definition with the fields: x, w.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4033.html#suggestion "Link to this heading")

Make sure to use the correct field identifiers instead:

```
fn main {
  struct S { x : Int; y : Int }
  let c = { x: 2, y: 1 }
}
```
