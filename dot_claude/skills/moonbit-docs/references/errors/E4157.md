# E4157[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4157.html#e4157 "Link to this heading")

Unsupported signature in mbti file.

Currently, only abstract type, functions and methods are supported in mbti file.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4157.html#erroneous-example "Link to this heading")

```
package "moonbit-community/E4157"

// Non abstract type is not supported
enum A {
  B
  C
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4157.html#suggestion "Link to this heading")

Use abstract type instead.

```
package "moonbit-community/E4157-fixed"

type A
```
