# E0057[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0057.html#e0057 "Link to this heading")

Constructor pattern expect payload.

The warning is emitted when the payload part is missing from the constructor
pattern.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0057.html#erroneous-example "Link to this heading")

```
///|
pub fn[T] is_some(s : T?) -> Bool {
  match s {
    Some => true
    None => false
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0057.html#suggestion "Link to this heading")

Use `(_)` to ignore all the payloads.

```
///|
pub fn[T] is_some(s : T?) -> Bool {
  match s {
    Some(_) => true
    None => false
  }
}
```
