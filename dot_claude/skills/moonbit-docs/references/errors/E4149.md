# E4149

## Contents

# E4149[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4149.html#e4149 "Link to this heading")

Cannot call async function in this context.

Async functions can only be called in async functions.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4149.html#erroneous-example "Link to this heading")

```
async fn f() -> Int {
  ...
}

fn g() -> Int {
  f!()
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4149.html#suggestion "Link to this heading")

You can change the containing function to an async function:

```
async fn g() -> Int {
  f!()
}
```
