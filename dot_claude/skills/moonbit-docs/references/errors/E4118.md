# E4118

## Contents

# E4118[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4118.html#e4118 "Link to this heading")

Cannot match type with map pattern.

You can use map patterns on custom types, as long as they have a method `op_get`
that returns an optional value for a given key.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4118.html#erroneous-example "Link to this heading")

```
type MyMap[K, V] Map[K, V]

fn main {
  let map : MyMap[String, Int] = { "a": 1, "b": 2, "c": 3 }
  match map {
    { "a": a, .. } => println("a: \{a}")
  //^~~~~~~~~~~~~~
  // Error: Please implement method `op_get` for type MyMap[String, Int] to match it with map pattern.
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4118.html#suggestion "Link to this heading")

Implement the `op_get` method as suggested in the error message.

```
pub fn MyMap::op_get[K: Eq + Hash, V](https://docs.moonbitlang.com/en/stable/language/error_codes/self : MyMap[K, V], key : K) -> V? {
  self._[key]
}
```
