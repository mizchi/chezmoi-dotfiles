# E3004[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3004.html#e3004 "Link to this heading")

Missing parameters list. Add `()` after the name of the function if it takes 0
parameter.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3004.html#erroneous-example "Link to this heading")

```
///|
/// Error: Missing parameters list. Add `()` if function `greet` has 0 parameter.
pub fn greet -> Unit {
  println("Hello, world!")
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3004.html#suggestion "Link to this heading")

Add `()` after the function name.

```
///|
pub fn greet() -> Unit {
  println("Hello, world!")
}
```
