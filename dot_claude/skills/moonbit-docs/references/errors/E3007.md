# E3007[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3007.html#e3007 "Link to this heading")

Wrong location of `..` in pattern match. Put `..` at the end of the pattern.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3007.html#erroneous-example "Link to this heading")

```
struct S {
  a : Int
  b : Int
  c : Int
}

fn main {
  let s : S = { a : 1, b : 2, c : 3 }
  let { a, .., c } = s
  //       ^^
  // Error: Unexpected `..` here, add `, ..` behind the last field to ignore the rest of struct.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3007.html#suggestion "Link to this heading")

```
// ...
fn main {
  // ...

}
```
