# E4064[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4064.html#e4064 "Link to this heading")

Invalid escape sequence in char/string literal.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4064.html#erroneous-example "Link to this heading")

Using escape sequence with invalid unicode code point result in this error:

```
test {
  println('\uD800')
}
```

Another case is using unsupported escape sequence in char literal for `Byte`:

```
test {
  let _ : Byte = '\uD000'
}
```

Only `\x` and `\o` escape sequences are supported in `Byte` literal.
