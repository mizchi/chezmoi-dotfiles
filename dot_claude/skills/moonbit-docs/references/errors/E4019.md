# E4019

## Contents

# E4019[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4019.html#e4019 "Link to this heading")

The label is declared twice in this function.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4019.html#erroneous-example "Link to this heading")

```
fn f(g~ : Int, g~ : String) -> Int {
  g
}
```

The above example declares the label `g` twice in the function `f`,
which is not allowed and gives the following error on line 1:

```
The label g~ is declared twice in this function, first in <FILE>.mbt:1:6
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4019.html#suggestion "Link to this heading")

Rename one of the labels to avoid the conflict:

```
fn f(g~ : Int, h~ : String) -> Int {
  g
}
```
