# E3016

## Contents

# E3016[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3016.html#e3016 "Link to this heading")

Unexpected `~` in argument. Did you mean `label=pattern` or `label~`?

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3016.html#erroneous-example "Link to this heading")

```
enum A {
  A(b~ : Int)
}

fn main {
  match A(b=1) {
    A(b~=c) => {
      println("Hello, world! \{c}") // Error: unexpected `~` in argument. Did you mean `label=pattern` or `label~`?
    }
  }
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3016.html#suggestion "Link to this heading")

Replace `b~` with `b=` or `b~` as appropriate.

```
enum A {
  A(b~ : Int)
}

fn main {
  match A(b=1) {
    A(b~) => {
      println("Hello, world! \{b}")
    }
  }
}
```

Or,

```
enum A {
  A(b~ : Int)
}

fn main {
  match A(b=1) {
    A(b=c) => {
      println("Hello, world! \{c}")
    }
  }
}
```
