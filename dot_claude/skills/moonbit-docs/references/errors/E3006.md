# E3006[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3006.html#e3006 "Link to this heading")

There is no individual visibility control for enum constructors.

Usually, this means that you put an `priv` or `pub` visibility modifier on a enum constructor.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3006.html#erroneous-example "Link to this heading")

```
enum A {
  priv A1 // Error: No individual visibility for enum constructor.
  pub A2 // Error: No individual visibility for enum constructor.
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3006.html#suggestion "Link to this heading")

Remove the visibility modifier from the definition of the enum constructor:

```
enum A {
  A1
  A2
}
```
