# E3020

## Contents

# E3020[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3020.html#e3020 "Link to this heading")

Unexpected `=` in struct expression. The correct syntax for struct expression is
`{ field: expression }`.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3020.html#erroneous-example "Link to this heading")

```
struct S {
  a : Int
  b : Int
}

fn main {
  let s : S = { a : 1, b : 2 }
  let ss = { ..s, a = 1 } // Error: Unexpected `=` in struct expression.
  ignore(ss)
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3020.html#suggestion "Link to this heading")

Change the struct expression to use the correct syntax:

```
struct S {
  a : Int
  b : Int
}

fn main {
  let s : S = { a : 1, b : 2 }
  let ss = { ..s, a : 1 }
  ignore(ss)
}
```
