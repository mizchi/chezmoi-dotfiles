# E4162

## Contents

# E4162[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4162.html#e4162 "Link to this heading")

Local parameter is not function.

`#locals` is an attribute that aims to inline the given function parameter. This
error is raised when the given parameter is not a function.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4162.html#erroneous-example "Link to this heading")

```
///|
#locals(init_)
pub fn[A, B] fold(a : Array[A], init_~ : B, f : (B, A) -> B) -> B {
  //                            ^
  // This parameter is expected to be a function, but it is not.
  let mut acc = init_
  for x in a {
    acc = f(acc, x)
  }
  acc
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4162.html#suggestion "Link to this heading")

Apply `locals` to local functions only.

```
///|
#locals(f)
pub fn[A, B] fold(a : Array[A], init_~ : B, f : (B, A) -> B) -> B {
  let mut acc = init_
  for x in a {
    acc = f(acc, x)
  }
  acc
}
```
