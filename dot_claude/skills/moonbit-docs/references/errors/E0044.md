# E0044[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0044.html#e0044 "Link to this heading")

Inline wasm code is invalid.

The inline wasm code refers to unbound function.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0044.html#erroneous-example "Link to this heading")

```
///|
extern "wasm" fn g() =
  #| (func $g (call $does_not_exist))
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0044.html#suggestion "Link to this heading")

If you know that such a function does exist, manually override suppress the
warning. Notice that the compilation error will result in an
[internal compiler error](https://docs.moonbitlang.com/en/stable/language/error_codes/E1001.html)
