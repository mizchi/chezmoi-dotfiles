# E0008[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0008.html#e0008 "Link to this heading")

The modifier (`pub`/`priv`) is redundant here, since this field has such
visibility by default.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0008.html#erroneous-example "Link to this heading")

```
///|
struct A {
  priv value : Int
  //^^
}

///|
pub struct B {
  pub value : Int
  //^
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E0008.html#suggestion "Link to this heading")

Remove the visibility modifier on the field.

```
///|
struct A {
  value : Int
}

///|
pub struct B {
  value : Int
}
```
