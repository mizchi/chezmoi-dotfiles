# E3010

## Contents

# E3010[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3010.html#e3010 "Link to this heading")

Only labelled arguments can have default value.

## Erroneous example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3010.html#erroneous-example "Link to this heading")

```
pub fn greet(name : String = "World") -> Unit {
  //                       ^ Error: Only labelled arguments can have default value.
  println("Hello, " + name + "!")
}
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E3010.html#suggestion "Link to this heading")

Use an optional argument with `?` if you want to provide a default value:

```
///|
pub fn greet(name? : String = "World") -> Unit {
  println("Hello, " + name + "!")
}
```

Or remove the default value if you want to keep it as a positional argument:

```
///|
pub fn greet_without_default(name : String) -> Unit {
  println("Hello, " + name + "!")
}
```
