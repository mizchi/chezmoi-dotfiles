# E4031

## Contents

# E4031[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4031.html#e4031 "Link to this heading")

The constructor is not found.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4031.html#erroneous-example "Link to this heading")

```
enum U { V }
let v : U = W
```

The example above tries to assign a variant `W` to a variable `v` of type `U`,
but this variant doesn't exist, giving the following error on line 2:

```
The variant type U does not have the constructor W.
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4031.html#suggestion "Link to this heading")

Make sure to use the correct constructor instead:

```
enum U { V }
let v : U = V
```
