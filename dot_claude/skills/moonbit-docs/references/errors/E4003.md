## Contents

# E4003[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4003.html#e4003 "Link to this heading")

This is a reserved type name. Cannot declare it as a type variable, type, or trait.

## Erroneous Example[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4003.html#erroneous-example "Link to this heading")

```
struct Error {}
```

`Error` is the reserved name for the built-in error type, so it cannot be used for a
custom type. This will give the following error on line 1:

```
"Error" is a reserved type name. Cannot declare it as type
```

## Suggestion[#](https://docs.moonbitlang.com/en/stable/language/error_codes/E4003.html#suggestion "Link to this heading")

Consider using another name for your type, such as `MyError` or `Error_`.

```
struct MyError {}
```
